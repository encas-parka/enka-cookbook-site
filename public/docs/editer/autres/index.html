Ajouter ingrédients, catégories, etc.

Des listes d’ingrédients, de matériel nécessaires à l’élaboration des recettes (four, gastro, etc.), et des catégories pour classer les recettes (tartes, snacking, soupe, etc.) sont déjà enregistrés dans le site et donc utilisables lorsque l’on édite/créé une recette. Mais si vous souhaitez en ajouter, c’est possible depuis l’interface d’édition (colonne de gauche).

Pour les ingrédients, il est demandé de préciser s’il est allergène, ou s’il contient de la viande ou du porc (ça permet d’ajouter automatiquement des alertes sur les recettes qui utilisent ces ingrédients).

→ ajouter un ingrédient → voir la liste d'ingrédients disponible → ajouter un materiel → ajouter une catégorie

Les ingrédients sont regroupés dans listes. Dans certaines d’entre-elles, il est possible de préciser pour les ingrédients une saison (ex: fruits et légumes), et s’il s’agit d’un ingrédient frais (pouvant nécessiter une conservation au frigo).
