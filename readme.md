# Enka Cookbook Site

## Présentation

Ce dépôt héberge un site permettant le partage de recettes de cuisine. Il est construit à partir du thème [hugo-cookbook-theme](https://github.com/encas-parka/hugo-cookbook-theme). Hugo sert ici principalement à la génération de json, consommés par une app svelte 5.

Enka cookbook est un outil pour les collectifs cuisinant à l'occasion d'événements militants/solidaires. Il permet notamment le partage de recettes de cuisine (avec calcul des quantités en fonction du nombre de couverts prévus), la création de menus et séries de menus pour des événements se déroulant sur plusieurs jours, la gestion du liste de course collaboratives, la reparition des tache.

## Organisation de cantines

Ces pages événements permettent:

- de rassembler différentes recettes pour le nombre de couverts prévus pour chaque repas / chaque jours.

- de lister l'ensemble des ingrédients nécessaires pour l'ensemble des recettes, avec le calcul des quantités totales pour chaque ingrédient. Il est possible d'y déclarer les produits acheté, et ainsi d'organiser l'approvisionnement.

### Impression et export PDF

Les pages de recettes, de menus et de listes de course sont optimisés pour des rendu d'impression propres (noir et blanc, saut de page automatique, boutons et menu de navigations masqué, etc.)
Une section permet d'imprime les menus pour un affichage destiné au mangeur•euses.
